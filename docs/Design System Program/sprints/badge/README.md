# Famille Badge

Primitive cible : `Badge` — MWI-COMP-317. Sa responsabilité est une étiquette statique, non interactive, rendue par un `span`. Les tons canoniques restent `cyan`, `violet`, `green`, `amber`, `red` et `neutral`.

Le pilote Kanban a migré deux libellés de catégorie et a validé la méthode sans régression. Le Sprint Famille validé inventorie le code courant et compose désormais `Badge` dans quatre wrappers métier à racine strictement équivalente, sans supprimer ces wrappers.

Ordre de lecture : `badge-family-inventory.md`, `badge-family-migration-plan.md`, puis `badge-family-sprint-report.md`.
