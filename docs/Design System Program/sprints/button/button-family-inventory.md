# Inventaire — famille Button

Date de baseline : 13 juillet 2026. Génération : AST TypeScript via `scripts/test-design-system-button-family.mjs --dump-inventory`.

## Méthode et totaux

L’inventaire couvre les usages directs de `Button`, les `<button>` natifs, les `role="button"`, les liens suffisamment stylés comme contrôles et les composants dont le nom contient Button, Action, Trigger, Toggle, Control ou Link. Il compte 291 sites : 92 canoniques, 184 natifs, 0 rôle non natif et 15 liens. Trente-neuf sites sont icon-only. Les catégories exclusives sont A=88, B=8, C=95 et D=100.

- A : usage générique déjà canonique, sûr et sans migration restante dans ce sprint ;
- B : wrapper métier composable, conservé ; seuls `ExternalButton` et `LoadMoreButton` sont candidats conditionnels ;
- C : faux générique — toggle, navigation, tab, carte, drag, modal ou contrôle riche, non migré ;
- D : ambigu ou legacy, documenté uniquement.
