## Lecture

Le placeholder est relevé comme source runtime lorsqu’il influence le calcul du navigateur, mais il n’est jamais accepté comme unique label conforme. Une valeur « — » dans la cohorte signifie que le navigateur n’expose aucun nom exploitable. Les cas B obtiennent uniquement une relation vers du texte métier existant ; les cas C et D sont volontairement laissés inchangés.
