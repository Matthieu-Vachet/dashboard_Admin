# COMP-331 — Navigation interne responsive Admin Pokémon

Sur desktop, une barre compacte combine recherche, groupes et sections du seul groupe actif. Sur mobile et tablette étroite, un bouton de section ouvre un panneau plein écran, verrouille le scroll, focalise la recherche et se ferme avec Échap. La section active est persistée dans `?section=`.
